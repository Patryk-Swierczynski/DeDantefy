# DeDantefy
Powyższy ko rekurencyjnie usuwa wszystkie pliki należące do listy plików dante oraz usuwa linie z includami z dante

### ! nazwy plików ze spacjami wybuchają projekt
Z jakiegoś powodu linia poleceń nie współgra dobrze ze spacjami także lepiej ich unikać bo projekt wybuchnie

### Czemu nie daję executable
Żeby nie było problemów że coś nielegalnego odbyło się w kodzie ,skompiluj sobie sam i zobacz czy działa.

### System
Projekt dobrze działa na windowsie , na linuksie również powinien działać dobrze ale go nie testowałem może w przyszłości to zrobię

# Jak korzystać
1. Wkleić kod z maina
2. Dodać w CLionie w profilu buildu ścieżkę do folderu z plikami/lub podfolderami z plikami
3. włączyć skrypt
4. cieszyć się czasem zaoszczędzonym na nie robieniu tego ręcznie
